Вопросы для собеседования на тему
Full Stack разработчик HTML + CSS

  1. Что такое SVG в HTML?

    HTML SVG используется для описания векторной или растровой графики. Изображения SVG и их поведение определяются в XML-файлах. Мы чаще всего используем его для векторных диаграмм, таких как круговые диаграммы, двумерные графики в системе координат X, Y.

  2. Что такое пустые элементы в HTML?

    Пустые элементы в HTML - это теги, которые не требуют закрывающего тега. Они используются для вставки изображений, переносов строк и другого контента, который не требует дополнительной информации.

  3. Что такое формы и как их создавать в HTML?

    Формы собирают данные пользователей, такие как информация для входа или поисковые запросы. Формы можно создавать с помощью тега <form>, а поля ввода, такие как текстовые поля и флажки, можно добавлять с помощью различных других тегов.

  4. Как можно центрировать элемент по горизонтали и вертикали?

    Для центрирования элемента по горизонтали можно использовать margin: 0 auto; вместе с заданным шириной. Для центрирования элемента по вертикали можно использовать display: flex; вместе с align-items: centre; в контейнере.

  5. Что такое CSS?

    CSS расшифровывается как Каскадные таблицы стилей. Это язык таблиц стилей, используемый для описания внешнего вида и форматирования документа, написанного на HTML или XML.

  6. Как добавить JavaScript на веб-страницу HTML?

    JavaScript используется для создания интерактивных и удобных для пользователя веб-страниц. Это язык сценариев, который позволяет взаимодействовать с определенными элементами на странице на основе ввода пользователя. Как и с CSS, существуют три основных способа включения JavaScript: встроенный, блок скрипта и внешний файл JavaScript.

  7. Как обрабатывать события в HTML?

    События могут быть обработаны с помощью JavaScript, который может быть включен в HTML документ с использованием тега <script>. Слушатели событий могут быть добавлены к HTML элементам с использованием метода addEventListener(), который позволяет выполнять пользовательский код в ответ на действия пользователя, такие как клики или нажатия клавиш.

  8. Как можно вертикально выровнять текст внутри элемента?

    Вы можете вертикально выровнять текст, используя свойство vertical-align. Однако это применяется только к элементам inline или table-cell. Для элементов уровня блока вы можете использовать CSS flexbox или CSS grid для достижения вертикального выравнивания.

  9. Как изменить цвет текста с помощью CSS?

    Вы можете изменить цвет текста, используя свойство color в CSS. Например, color: red; сделает текст красным.

  10. Как можно выбрать элементы на основе их класса и идентификатора?

    Чтобы выбрать элементы на основе класса, вы добавляете префикс имени класса точкой (.) например .my-class. Чтобы выбрать элементы на основе идентификатора, вы добавляете префикс имени идентификатора решеткой (#) например #my-id.

  11. Что такое атрибут alt в HTML?

    Атрибут alt используется для отображения текста вместо изображения, когда изображение не может быть загружено из-за технической проблемы.

  12. Как создать ссылки на различные разделы на одной и той же веб-странице HTML?

    Мы используем тег <a>, а также ссылки с помощью символа #, чтобы создать несколько ссылок на различные разделы на одной и той же веб-странице.

  13. В чем разница между display: inline и display: inline-block?

    Свойство display: inline делает элемент похожим на элемент в строке, позволяя другим элементам появляться рядом на той же строке. display: inline-block также позволяет элементам появляться в строке, но сохраняет свойства блочного уровня, такие как установка ширины и высоты.
