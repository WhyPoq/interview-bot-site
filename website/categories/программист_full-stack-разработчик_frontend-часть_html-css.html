Full Stack разработчик HTML + CSS

  1. В чем разница между “display: none” и “visibility: hidden” при использовании как атрибуты для элемента HTML?

    Основное различие между “display: none” и “visibility: hidden” заключается в том, что первое удаляет элемент из потока документа, в то время как второе просто скрывает его. Элементы с “display: none” не видны и не занимают места на странице, в то время как элементы с “visibility: hidden” не видны, но все еще занимают место.

  2. Что такое белое пространство в HTML?

    Пустая последовательность пробельных символов называется белым пространством в HTML. Это белое пространство считается одним пробельным символом в HTML. Белое пространство помогает браузеру объединять несколько пробелов в один, что упрощает работу с отступами. Белое пространство помогает лучше организовать содержимое и теги, делая их более читаемыми и понятными.

  3. Как работает CSS анимация и какие существуют свойства, связанные с анимацией?

    CSS анимация позволяет анимировать элементы без необходимости использования JavaScript. Свойство animation определяет ключевые кадры анимации, продолжительность, функцию времени, задержку и количество итераций. Другие свойства, связанные с анимацией, включают @keyframes, animation-timing-function, animation-delay и animation-iteration-count.

  4. Что такое правило CSS @media и как оно используется?

    Правило @media используется для применения конкретных правил CSS на основе характеристик устройства вывода. Оно позволяет определять различные стили для различных типов медиа (например, экран, печать) или конкретных функций (например, экраны высокого разрешения).

  5. Как мы можем вставить комментарий в HTML?

    Мы можем вставить комментарий в HTML, начиная с знака меньше и заканчивая знаком больше. Например, "<!-" и "->".

  6. Как добавить цвет к тексту в HTML?

    Вы можете добавить цвет к тексту в HTML, используя следующий код: <!DOCTYPE html> <html> <body> <h2>Пример цветного текста в HTML</h2> <h1 style="color: Red">Привет HTML</h1> <p style="color: Blue">Строка 1</p> <p style="color: Green">Строка 2</p> </body> </html>

  7. Что такое SVG в HTML?

    HTML SVG используется для описания векторной или растровой графики. Изображения SVG и их поведение определяются в XML-файлах. Мы чаще всего используем его для векторных диаграмм, таких как круговые диаграммы, двумерные графики в системе координат X, Y.

  8. В чем разница между встроенным, внутренним и внешним CSS?

    Встроенный CSS применяется непосредственно внутри HTML элементов с использованием атрибута style. Внутренний CSS определяется внутри тегов <style> в разделе <head> HTML. Внешний CSS хранится в отдельных файлах CSS и связывается с HTML документом с помощью элемента <link>.

  9. Что такое препроцессоры CSS, и зачем они полезны?

    Препроцессоры CSS, такие как Sass, Less и Stylus, являются расширениями CSS, которые предоставляют дополнительные функции, такие как переменные, миксины, вложенность и функции. Они улучшают организацию кода, повторное использование и упрощают поддержку больших кодовых баз CSS.

  10. Как добавить кнопки в HTML?

    Мы можем использовать встроенный тег Button в HTML для добавления кнопок на веб-страницу HTML.

  11. Какая иерархия стилевых таблиц?

    Если один селектор включает три различных определения стилей, то определение, ближайшее к фактическому тегу, имеет приоритет. Встроенный стиль имеет приоритет над внедренными таблицами стилей, которые имеют приоритет над внешними таблицами стилей.

  12. Что такое формы и как их создавать в HTML?

    Формы собирают данные пользователей, такие как информация для входа или поисковые запросы. Формы можно создавать с помощью тега <form>, а поля ввода, такие как текстовые поля и флажки, можно добавлять с помощью различных других тегов.

  13. Как включить CSS на веб-странице?

    CSS можно включить на веб-странице, используя элемент <link> в разделе HTML <head> или непосредственно с помощью атрибута style в HTML-элементах.

  14. Какие существуют различные типы заголовков в HTML?

    Существует шесть типов тегов заголовков в HTML, которые определяются с помощью тегов <h1> до <h6>. Каждый тип тега заголовка отображает текст разного размера. <h1> - самый крупный тег заголовка, а <h6> - самый маленький.

  15. Что такое селекторы CSS?

    Селекторы CSS - это шаблоны, используемые для выбора и стилизации конкретных элементов в документе HTML. Примеры селекторов включают селекторы элементов (p, div), селекторы классов (.my-class) и селекторы идентификаторов (#my-id).
