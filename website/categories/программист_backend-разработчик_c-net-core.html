Вопросы для собеседования на тему

Backend разработчик C# + .NET Core

  1. Что такое класс Hash table в C#?

    Класс Hash table представляет собой коллекцию пар ключ/значение, организованных на основе хэш-кода ключа.

  2. В чем разница между directcast и ctype?

    Если объект должен иметь тип времени выполнения, аналогичный другому объекту, то используется DirectCast для преобразования. Когда преобразование происходит между выражением и типом, то используется Ctype.

  3. Что произойдет, если имена методов в унаследованных интерфейсах конфликтуют?

    Проблема может возникнуть, когда методы из различных интерфейсов ожидают разные данные. Но когда речь идет о самом компиляторе, проблем не должно возникнуть.

  4. Можно ли использовать команду “this” в статическом методе?

    Нет. Это потому что только статические переменные/методы могут быть использованы в статическом методе.

  5. Как наследовать класс в другой класс в C#?

    В C#, двоеточие можно использовать как оператор наследования. Вам нужно поставить двоеточие и следовать за ним именем класса.

  6. Что такое мультикастинг делегатов в C#?

    Мультикастинг делегатов позволяет пользователям указывать на более чем один метод в одном вызове.

  7. Обсудите различные методы передачи параметров в методе.

    Различные методы передачи параметров в метод включают в себя - Выходные параметры: Позволяют методу возвращать более одного значения. Параметры значений: Формальное значение копирует и сохраняет значение фактического аргумента, что позволяет манипулировать формальным параметром без влияния на значение фактического параметра. Параметры ссылки: Адрес памяти фактического параметра сохраняется в формальном аргументе, что означает, что любое изменение формального параметра будет отражаться и на фактическом аргументе.

  8. Что такое конструкторы?

    Конструктор - это функция-член с тем же именем, что и его класс. Конструктор автоматически вызывается при создании объекта. При инициализации класса он конструирует все значения членов данных.

  9. В чем разница между параметрами out и ref?

    Когда аргумент передается как ref, он должен быть инициализирован перед передачей в метод. Параметр out, с другой стороны, не требуется инициализировать перед передачей в метод.

  10. Что вы понимаете под свойствами Get и Set Accessor?

    В C# Get и Set называются аксессорами, потому что они используют свойства. Такие закрытые поля доступны через аксессоры.

  11. В чем разница между read-only и константами?

    Во время компиляции константные переменные объявляются и инициализируются. Позже невозможно изменить это конкретное значение. С другой стороны, read-only используется после присвоения значения во время выполнения.

  12. Что такое объект?

    Объект - это экземпляр класса, который можно использовать для доступа к методам класса. Ключевое слово 'New' можно использовать для создания объекта.

  13. Как можно отсортировать элементы массива в порядке убывания?

    Вы можете использовать метод Sort() и затем метод Reverse().

  14. В чем разница между операторами “is” и “as” в C#?

    Оператор “is” может использоваться для проверки совместимости объекта с заданным типом, и результат возвращается как логическое значение. Оператор “as” может использоваться для приведения объекта к типу или классу.

  15. Напишите особенности обобщений в C#?

    Обобщения - это техника улучшения вашей программы различными способами, включая создание обобщенных классов и повторное использование кода.
