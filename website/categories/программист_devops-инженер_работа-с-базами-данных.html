Вопросы для собеседования на тему
DevOps инженер Работа с базами данных

  1. Объясните Нормализацию и Денормализацию.

    Нормализация - это процесс удаления избыточных данных из базы данных путем разделения таблицы в хорошо определенном порядке для поддержания целостности данных. Этот процесс экономит много места для хранения. Денормализация - это процесс добавления избыточных данных в таблицу для ускорения выполнения сложных запросов и, таким образом, достижения лучшей производительности.

  2. Что такое RDBMS?

    Система управления реляционными базами данных (RDBMS) основана на реляционной модели данных, которая хранится в базах данных в отдельных таблицах и связана с использованием общего столбца. Данные могут быть легко получены из реляционной базы данных с использованием языка структурированных запросов (SQL).

  3. Объясните Словарь данных.

    Словарь данных - это набор информации, описывающий содержание и структуру таблиц и объектов базы данных. Задача информации, хранящейся в словаре данных, заключается в управлении, манипулировании и доступе к отношениям между элементами базы данных.

  4. Перечислите некоторые команды DDL, DML и DCL.

    Команды языка определения данных (DDL): CREATE, ALTER, TRUNCATE, DROP, RENAME. Команды языка манипулирования данными (DML): INSERT, UPDATE, DELETE, MERGE. Команды языка управления данными (DCL): COMMIT, ROLLBACK, SAVEPOINT.

  5. Что такое хранилище данных?

    Хранение и доступ к данным, полученным из транзакций и других источников, из центрального места для проведения анализа, называется хранилищем данных.

  6. Когда функциональная зависимость считается полностью функционально зависимой?

    Для того чтобы соответствовать критериям полной функциональной зависимости, отношение должно удовлетворять требованию функциональной зависимости. Функциональная зависимость 'A' и 'B' считается полностью функционально зависимой, когда удаление любого атрибута, скажем 'X', из 'A' означает, что зависимость больше не существует.

  7. Что такое фантомная блокировка?

    Обнаружение фантомной блокировки - это состояние, когда блокировка на самом деле не существует, но из-за задержки в распространении локальной информации алгоритмы обнаружения блокировок идентифицируют блокировки.

  8. Что такое разделение баз данных?

    Разделение баз данных - это процесс разделения таблиц, индексов на более мелкие части для управления и доступа к данным на более тонком уровне. Этот процесс разделения снижает стоимость хранения большого объема данных, а также улучшает производительность и управляемость.

  9. Что такое SQL?

    Структурированный язык запросов, SQL - это стандартный язык программирования ANSI (American National Standard Institute), который специально разработан для хранения и управления данными в системе управления реляционными базами данных (RDBMS) с использованием всех видов операций с данными.

  10. Объясните Первичный ключ и Составной ключ.

    Первичный ключ - это столбец таблицы, данные каждой строки которого уникально идентифицируются. В каждой строке таблицы должен быть первичный ключ, и ни две строки не могут иметь одинаковый первичный ключ. Значение первичного ключа никогда не может быть пустым или изменено. Составной ключ - это форма кандидатского ключа, где набор столбцов уникально идентифицирует каждую строку в таблице.

  11. Что вы понимаете под B-деревьями?

    B-дерево представляет структуру данных в виде дерева для внешней памяти, которая читает и записывает большие блоки данных. Оно часто используется в базах данных и файловых системах, где все операции вставки, удаления, сортировки и т. д. выполняются за логарифмическое время.

  12. Что такое хранимые процедуры?

    Хранимая процедура - это набор предварительно скомпилированных SQL-запросов, которые при выполнении обозначают программу, принимающую ввод, обрабатывающую и выдающую вывод.

  13. Определите курсор и его типы.

    Курсор - это временная рабочая область, которая хранит данные, а также набор результатов, возникших после манипуляции извлеченными данными. Курсор может содержать только одну строку. Существуют 2 типа курсоров: Неявные курсоры объявляются автоматически при выполнении операторов DML, таких как INSERT, UPDATE, DELETE. Явные курсоры должны быть объявлены при выполнении операторов SELECT, возвращающих более одной строки.

  14. Что такое VDL и SDL?

    VDL - это Язык Определения Представлений, который представляет представления пользователей и их сопоставление с концептуальной схемой. SDL - это Язык Определения Хранилища, который определяет сопоставление между двумя схемами.
