Вопросы для собеседования на тему

Full Stack разработчик C# + .NET Core

  1. Напишите особенности обобщений в C#?

    Обобщения - это техника улучшения вашей программы различными способами, включая создание обобщенных классов и повторное использование кода.

  2. В чем разница между System.Array.CopyTo() и System.Array.Clone()?

    В методе Clone() создается новый массив объектов со всеми оригинальными элементами массива, используя метод CopyTo(). По сути, все элементы, присутствующие в существующем массиве, копируются в другой существующий массив.

  3. Какие существуют классы ввода-вывода в C#?

    В C# пространство имен System.IO содержит несколько классов, которые используются для выполнения различных операций с файлами, таких как создание, удаление, закрытие и открытие.

  4. Что такое консольное приложение?

    Приложение, которое может запускаться в окне командной строки, называется консольным приложением.

  5. Что такое перегрузка методов?

    Перегрузка методов - это процесс создания множества методов в одном классе с одинаковым именем, но различными сигнатурами. Компилятор использует разрешение перегрузки, чтобы определить, какой метод вызвать при компиляции.

  6. Что вы понимаете под свойствами Get и Set Accessor?

    В C# Get и Set называются аксессорами, потому что они используют свойства. Такие приватные поля доступны через аксессоры.

  7. Укажите основные особенности C# кратко.

    Некоторые из основных особенностей C# - C# является языком с безопасной типизацией и управляемым кодом. C# ориентирован на объекты. C# - дружелюбный кроссплатформенный язык. C# - платформонезависимый язык при компиляции. C# - общего назначения. C# используется для реализации деструкторов и конструкторов. C# является частью .NET-фреймворка. C# - легкий для изучения и понимания язык. C# - структурированный язык.

  8. Как использовать nullable типы в .Net?

    Когда типы значений могут принимать как обычные значения, так и значение null, их называют nullable типами.

  9. В чем разница между read-only и константами?

    Во время компиляции константные переменные объявляются и инициализируются. Позже невозможно изменить это конкретное значение. С другой стороны, read-only используется после присвоения значения во время выполнения.

  10. В чем разница между переопределением метода и перегрузкой метода?

    При переопределении метода соответствующее определение метода заменяется в производном классе, что изменяет поведение метода. Когда речь идет о перегрузке метода, создается метод с тем же именем и находится в том же классе, но имеет разные сигнатуры.

  11. Что такое типы значений и ссылочные типы?

    Тип значения содержит значение данных внутри своего пространства памяти. Ссылочный тип, с другой стороны, хранит адрес объекта, где хранится значение. Это, по сути, указатель на другое местоположение в памяти.

  12. Что такое объект?

    Объект - это экземпляр класса, который можно использовать для доступа к методам класса. Ключевое слово 'New' можно использовать для создания объекта.

  13. В чем разница между SortedList и SortedDictionary в C#?

    SortedList - это коллекция пар значений, отсортированных по ключам. SortedDictionary - это коллекция для хранения пар значений в отсортированной форме, при этом сортировка происходит по ключу.

  14. В чем разница между параметрами out и ref?

    Когда аргумент передается как ref, он должен быть инициализирован перед передачей в метод. Параметр out, с другой стороны, не требуется инициализировать перед передачей в метод.

  15. Что произойдет, если имена методов в унаследованных интерфейсах конфликтуют?

    Проблема может возникнуть, когда методы из различных интерфейсов ожидают разные данные. Но когда речь идет о самом компиляторе, проблем не должно возникнуть.
