Backend разработчик MySQL

  1. Что такое MySQL?

    MySQL - это система управления базами данных для веб-серверов. Она может масштабироваться вместе с веб-сайтом, так как она очень масштабируема. Большинство современных веб-сайтов работают на MySQL.

  2. Сколько триггеров возможно в MySQL?

    В MySQL можно использовать шесть триггеров: Before Insert, After Insert, Before Update, After Update, Before Delete, After Delete

  3. Что такое Sharding в SQL?

    Процесс разбиения больших таблиц на более мелкие фрагменты (называемые шардами), которые распределяются по нескольким серверам, называется Sharding. Преимущество Sharding заключается в том, что поскольку шардированная база данных обычно намного меньше оригинала, запросы, обслуживание и все другие задачи выполняются намного быстрее.

  4. Как создать и выполнить представления в MySQL?

    Создание представления выполняется с помощью оператора CREATE VIEW.

  5. Как удалить столбец из базы данных?

    Вы можете удалить столбец, используя ключевое слово DROP: ALTER TABLE classics DROP pages;

  6. Как создать базу данных в MySQL?

    Используйте следующую команду для создания новой базы данных с именем 'books': CREATE DATABASE books;

  7. Какие типы отношений используются в MySQL?

    Существует три категории отношений в MySQL: Один-к-Один, Один-ко-Многим и Многие-к-Многим.

  8. Что содержит база данных MySQL?

    База данных MySQL содержит одну или несколько таблиц, каждая из которых содержит записи или строки. Внутри этих строк находятся различные столбцы или поля, содержащие сами данные.

  9. Какие существуют типы данных для времени в MySQL?

    DATE, TIME, DATETIME, TIMESTAMP, YEAR

  10. Как можно просмотреть базу данных в MySQL?

    Чтобы просмотреть все базы данных на сервере MySQL, можно использовать следующую команду: mysql> SHOW DATABASES;

  11. Можете ли вы объяснить логическую архитектуру MySQL?

    Верхний уровень содержит службы, необходимые для большинства сетевых клиент-серверных инструментов или серверов, такие как обработка соединений, аутентификация, безопасность и т. д. Второй уровень содержит большую часть интеллекта MySQL. Здесь находится код для анализа, оптимизации, кэширования запросов и всех встроенных функций. Третий уровень содержит хранилища данных, отвечающие за хранение и извлечение данных, хранящихся в MySQL.

  12. Как удалить данные из таблицы MySQL?

    В MySQL для удаления записей из таблицы используется оператор DELETE: DELETE FROM table_name WHERE column_name = value_name

  13. Что такое запросы к базе данных MySQL?

    Запрос - это конкретный запрос или вопрос. С помощью запроса можно получить определенную информацию из базы данных и получить запись.

  14. Как вставить данные в MySQL?

    Оператор INSERT INTO используется для добавления новых записей в таблицу MySQL.

  15. Что такое сервер MySQL?

    Сервер, mysqld, является центром установки MySQL; он выполняет все манипуляции с базами данных и таблицами.
