Backend разработчик SQL Server

  1. Что такое пользовательская функция в SQL Server?

    Пользовательская функция позволяет пользователям писать свою логику по мере необходимости. Преимущество заключается в том, что она не ограничивается предопределенными функциями и позволяет писать простой или сложный SQL-код. Тип возвращаемого значения может быть таблицей или скалярным значением.

  2. Когда записи могут быть удалены из представления в SQL Server?

    Записи могут быть удалены из 'простого' представления, так как оно содержит данные только из одной таблицы.

  3. Каково значение CHECK в SQL Server?

    Ограничение CHECK ограничивает значения, которые могут быть помещены в столбец таблицы. Это поддерживает целостность данных. Ограничение используется по столбцам для задания конкретных значений этому столбцу. Пример: CONSTRAINT CHK_Student CHECK (age<20)

  4. Когда следует использовать курсоры, основанные на сервере?

    Когда вам нужно работать с одной записью в любой момент времени, вместо того чтобы брать все данные из таблицы как пакет. Производительность курсоров страдает, когда присутствуют большие объемы данных.

  5. Что означает столбец идентификатора в SQL?

    В SQL столбец идентификатора генерирует числовые значения автоматически. Эти столбцы не обязательно индексировать, и мы можем определить начальное и инкрементное значение столбца идентификатора.

  6. Что такое TCL-команды? и перечислите TCL-команды, доступные в SQL Server?

    TCL или команды языка управления транзакциями используются для управления различными транзакциями, происходящими в базе данных. Три команды TCL: Rollback, Save Tran и Commit.

  7. Расскажите шаги по скрытию экземпляров SQL Server.

    Для скрытия экземпляров SQL Server необходимо внести изменения в Менеджере конфигурации SQL Server, и для запуска этого требуется выполнить следующие шаги: Выбрать экземпляр SQL сервера, Выбрать свойства после щелчка правой кнопкой мыши, Установить Скрыть экземпляры на Да и нажать ПРИМЕНИТЬ, После внесения изменений перезапустить экземпляр SQL Server

  8. Укажите запрос, который возвращает список триггеров в базе данных.

    Select * from sys.objects where type='tr'

  9. Объясните команду DBCC и ее использование.

    Команда Database Consistency Checker (DBCC) проверяет целостность базы данных; Она помогает в проверке и мониторинге обслуживания базы данных, таблиц и проверке операций.

  10. Какую команду можно использовать для получения версии SQL Server?

    Select SERVERPROPERTY('productversion')

  11. Как создать таблицу в SQL?

    Для создания таблицы в SQL используется следующий запрос: Create table name_of_table( column1 datatype, column2 datatype ) Например: create table Student ( Name varchar(20), DOB date, Marks nvarchar(5), Subject varchar(20) )

  12. Расскажите о применении UPDATE STATISTICS.

    UPDATE STATISTICS используется для обновления информации о распределении ключевых значений для одной или нескольких групп/коллекций статистики в индексированном представлении или указанной таблице.
