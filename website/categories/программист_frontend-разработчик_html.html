Вопросы для собеседования на тему
Frontend разработчик HTML

  1. Что такое HTML теги?

    Мы используем HTML теги для размещения элементов в правильном и соответствующем формате. Теги используют символы < и >, чтобы отделить их от содержимого HTML. HTML теги не всегда нужно закрывать. Например, в случае изображений закрывающие теги не требуются, как в случае с тегом <img>.

  2. Как вставить символ копирайта в HTML?

    Вы можете вставить символ копирайта, используя &copy; или &#169; в файле HTML.

  3. Какова роль тега <head> в HTML?

    Тег <head> определяет информацию о веб-странице, которая не отображается на самой странице, такую как ее заголовок, ключевые слова и другие метаданные. Он расположен между тегами <html> и <body> и обычно является первым элементом в документе.

  4. Какая иерархия стилевых таблиц?

    Если один селектор включает три различных определения стилей, то определение, ближайшее к фактическому тегу, имеет приоритет. Встроенный стиль имеет приоритет над внедренными таблицами стилей, которые имеют приоритет над внешними таблицами стилей.

  5. Что такое атрибут 'class' в HTML?

    Атрибут 'class' в HTML определяет класс для элемента HTML. Он может использоваться для применения определенного стиля к группе элементов на веб-странице.

  6. Как вы разделяете секцию текста в HTML?

    Мы разделяем секцию текста в HTML, используя следующие теги: тег <br> – он используется для разделения строки текста. Он разрывает текущую строку и переносит поток текста на новую строку. Тег <p> – этот тег используется для написания абзаца текста. Тег <blockquote> – этот тег используется для определения больших цитируемых секций.

  7. Как вставить изображение на веб-страницу HTML?

    Вы можете вставить изображение на веб-страницу HTML, используя следующий код: <!DOCTYPE html> <html> <body> <h2>Пример изображения HTML</h2> <img src='tulip.jpeg' /> </body> </html>

  8. В чем разница между HTML и XHTML?

    HTML и XHTML - это обе языки разметки, используемые для создания веб-страниц. Однако XHTML строже, чем HTML, и требует от разработчиков писать хорошо структурированный код, который соответствует определенным правилам и рекомендациям. XHTML также требует закрытия всех тегов и заключения всех атрибутов в кавычки.

  9. Какие существуют типы списков в HTML?

    В HTML существуют три типа списков: упорядоченные, неупорядоченные и определения. Упорядоченные списки - это нумерованные списки, неупорядоченные списки - маркированные списки, а списки определений - списки терминов и их определений.

  10. Какова роль тега <meta> в HTML?

    Тег <meta> предоставляет дополнительную информацию о веб-странице, такую как автор, описание и ключевые слова. Он находится внутри раздела <head> документа HTML.

  11. Что произойдет, если между тегами HTML нет текста?

    Если между тегами отсутствует текст, то не будет ничего для форматирования. Следовательно, ничего не появится на экране. Некоторые теги, такие как теги без закрывающего тега, например тег <img>, не требуют наличия текста между ними.

  12. В чем разница между HTML и HTML5?

    HTML5 - последняя версия HTML и включает в себя новые функции и улучшения по сравнению с предыдущими версиями. Некоторые ключевые различия между HTML и HTML5 включают поддержку мультимедийных элементов (таких как видео и аудио), улучшенную семантику и лучшую поддержку мобильных устройств.

  13. В чем разница между Упорядоченным списком и Неупорядоченным списком?

    Неупорядоченный список использует теги <ul> </ul>, и каждый элемент списка написан между тегами <li> </li>. Элементы списка отображаются в виде маркеров, а не чисел. Упорядоченный список использует теги <ol> </ol>, и каждый элемент списка написан между тегами <li> </li>. Элементы списка отображаются в виде чисел, а не маркеров.
