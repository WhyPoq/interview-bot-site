Вопросы для собеседования на тему
Frontend разработчик HTML

  1. Как добавить стили CSS в HTML?

    Существует три способа включения CSS в HTML: встроенные стили CSS, внешние таблицы стилей и внутренние таблицы стилей.

  2. Какова роль атрибута method в формах HTML?

    Атрибут method используется для указания HTTP-метода, который будет использоваться для отправки данных формы. Два наиболее распространенных метода - GET и POST. GET используется для получения данных с сервера, в то время как POST используется для отправки данных на сервер

  3. В чем разница между HTML и HTML5?

    HTML5 - последняя версия HTML и включает в себя новые функции и улучшения по сравнению с предыдущими версиями. Некоторые ключевые различия между HTML и HTML5 включают поддержку мультимедийных элементов (таких как видео и аудио), улучшенную семантику и лучшую поддержку мобильных устройств.

  4. Как добавить JavaScript на веб-страницу HTML?

    JavaScript используется для создания интерактивных и удобных для пользователя веб-страниц. Это язык сценариев, который позволяет взаимодействовать с определенными элементами на странице на основе ввода пользователя. Как и с CSS, существуют три основных способа включения JavaScript: встроенный, блок скрипта и внешний файл JavaScript.

  5. Что такое семантический HTML?

    Семантический HTML - это стиль кодирования. Это использование разметки HTML для усиления семантики или значения содержимого.

  6. Как добавить кнопки в HTML?

    Мы можем использовать встроенный тег Button в HTML для добавления кнопок на веб-страницу HTML.

  7. Как вы разделяете секцию текста в HTML?

    Мы разделяем секцию текста в HTML, используя следующие теги: тег <br> – он используется для разделения строки текста. Он разрывает текущую строку и переносит поток текста на новую строку. Тег <p> – этот тег используется для написания абзаца текста. Тег <blockquote> – этот тег используется для определения больших цитируемых секций.

  8. В чем разница между абсолютным и относительным URL?

    Абсолютный URL включает полный веб-адрес, протокол (например, http или https) и доменное имя (например, www.example.com). Относительный URL указывает местоположение ресурса относительно текущей веб-страницы.

  9. Какова роль атрибута title в HTML?

    Атрибут title предоставляет дополнительную информацию об элементе, таком как ссылка или изображение. Текст заголовка отображается в виде всплывающей подсказки, когда пользователь наводит курсор на элемент.

  10. Когда использовать скрипты в заголовке и когда использовать скрипты в теле?

    Скрипты можно разместить в разделе <head> документа HTML или в разделе <body>. Скрипты, которые должны быть выполнены до отображения страницы, такие как скрипты, которые определяют переменные или функции, должны быть размещены в разделе <head>. Скрипты, которые должны быть выполнены после отображения страницы, такие как скрипты, которые манипулируют DOM, должны быть размещены в разделе <body>.

  11. Что такое атрибут 'class' в HTML?

    Атрибут 'class' в HTML определяет класс для элемента HTML. Он может использоваться для применения определенного стиля к группе элементов на веб-странице.

  12. Какими способами можно отображать элементы HTML?

    Элементы HTML можно отображать несколькими способами, включая блочный, строчный, строчно-блочный и ни один. Свойство display может указать, как элемент должен быть отображен.

  13. Являются ли HTML теги и элементы одним и тем же?

    Нет, HTML теги используются для определения структуры веб-страницы, в то время как HTML элементы состоят из набора тегов, которые определяют конкретную часть веб-страницы.

  14. Какие существуют различные типы заголовков в HTML?

    Существует шесть типов тегов заголовков в HTML, которые определяются с помощью тегов <h1> до <h6>. Каждый тип тега заголовка отображает текст разного размера. <h1> является самым крупным тегом заголовка, а <h6> - самым маленьким.

  15. Что произойдет, если между тегами HTML нет текста?

    Если между тегами отсутствует текст, то не будет ничего для форматирования. Следовательно, ничего не появится на экране. Некоторые теги, такие как теги без закрывающего тега, например тег <img>, не требуют наличия текста между ними.
