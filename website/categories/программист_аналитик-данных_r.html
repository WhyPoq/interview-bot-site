Аналитик данных R

  1. Для чего используется пакет MASS?

    Функции MASS включают те функции, которые выполняют анализ линейных и квадратичных дискриминантных функций.

  2. Что такое функция cv.lm()?

    Она определена в пакете DAAG, который используется для кросс-валидации.

  3. Какова полная форма SEM?

    SEM расшифровывается как Структурное уравнение моделирования.

  4. Для чего используется функция boot.sem()?

    Она используется для бутстрэпа структурной модели уравнений.

  5. В чем разница между lapply() и sapply()?

    lapply() используется для вывода результатов в виде списка, в то время как sapply() используется для вывода результатов в виде вектора или фрейма данных

  6. Какая команда используется для восстановления объекта R из файла?

    Команда load используется для сохранения объектов R из файла. Синтаксис: >load(”z.Rdata”)

  7. Какой пакет предоставляет бутстраппинг?

    Используется пакет boot, который предоставляет бутстраппинг.

  8. Сколько существует алгоритмов сортировки?

    Используются 5 типов алгоритмов сортировки, которые включают в себя: сортировку пузырьком, сортировку выбором, сортировку слиянием, быструю сортировку, сортировку корзины

  9. Что такое lda()?

    lda() используется для печати дискриминантных функций, которые основаны на центрированных переменных.

  10. Что такое регрессия Пуассона?

    Она используется для прогнозирования переменной результата, которая представляет собой счета из заданного набора непрерывных предикторов.

  11. Что такое R?

    R - это язык программирования, который используется для разработки статистического программного обеспечения и анализа данных. Его также все чаще применяют для приложений машинного обучения.

  12. Что такое FactoMineR?

    Это пакет, который включает в себя количественные и качественные переменные. Он также включает дополнительные переменные и наблюдения.

  13. Что такое рабочее пространство в R?

    Рабочее пространство - это текущая рабочая среда R, которая включает в себя любые определенные пользователем объекты, такие как векторы, списки и т. д.

  14. Каков полный формат CFA?

    CFA расшифровывается как Confirmatory Factor Analysis.

  15. Для чего используется пакет doBY?

    Он используется для определения желаемой таблицы с использованием функции и модельной формулы.
