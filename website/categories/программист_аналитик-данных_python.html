Аналитик данных Python

  1. Что такое модули Python? Назовите несколько встроенных модулей Python, которые часто используются.

    Модули Python - это файлы, содержащие код Python. В этом коде могут использоваться функции, классы или переменные. Модуль Python - это файл .py, содержащий код, который может быть выполнен. Вот несколько часто используемых встроенных модулей: JSON, data time, random, math, sys, OS.

  2. Как найти позиции чисел, которые являются кратными трём из серии?

  3. В Python аргументы передаются по значению или по ссылке?

    Аргументы передаются по ссылке в Python.

  4. Что такое словарь в Python?

    Словарь - один из встроенных типов данных Python. Он устанавливает однозначное соответствие между ключами и значениями. Ключи и значения словаря хранятся парами в словарях. Ключи используются для индексации словарей.

  5. Существует ли встроенный цикл do-while в Python?

    Нет

  6. В чем разница между поверхностным копированием и глубоким копированием?

    Deepcopy создает другой объект и заполняет его дочерними объектами исходного объекта. Поэтому изменения в исходном объекте не отражаются в копии. Поверхностное копирование создает другой объект и заполняет его ссылками на дочерние объекты внутри исходного объекта. Поэтому изменения в исходном объекте отражаются в копии.

  7. Что такое PEP 8?

    PEP обозначает Предложение по улучшению Python. Это сборник рекомендаций по форматированию кода Python для максимальной читаемости.

  8. Как в Python управляется память?

    Управление памятью в Python осуществляется частным кучей. Частная куча содержит все объекты Python и структуры данных. Эта секретная куча недоступна программисту. Вместо этого об этом заботится интерпретатор Python. Python также включает в себя встроенный сборщик мусора, который перерабатывает всю неиспользуемую память и делает ее доступной для кучи. Управление памятью в Python отвечает за выделение кучи для объектов Python. Основной API предоставляет программистам доступ к некоторым инструментам программирования.

  9. Как вы будете делать первую букву строки заглавной?

    Функция capitalize() в Python делает первую букву строки заглавной. Она возвращает исходный текст, если строка уже содержит заглавную букву в начале.

  10. Используются ли в Python спецификаторы доступа?

    Доступ к переменной экземпляра или функции в Python не ограничен. Чтобы имитировать поведение защищенных и закрытых спецификаторов доступа, Python вводит идею префиксации имени переменной, функции или метода одинарным или двойным подчеркиванием.

  11. Как написать Unicode строку в Python?

    Старый тип Unicode был заменен на тип 'str' в Python 3, и строка теперь по умолчанию считается Unicode. Используя функцию art.title.encode('utf-8'), мы можем создать Unicode строку.

  12. Объясните пакеты Python.

    Пакеты в Python - это пространства имен, которые содержат множество модулей.

  13. Что такое массив Numpy?

    Массив numpy - это сетка значений, все одного типа, и индексируется кортежем неотрицательных целых чисел. Количество измерений определяет ранг массива. Форма массива - это кортеж целых чисел, указывающий размер массива вдоль каждого измерения.

  14. Каков вывод: Print(__name__)? Обоснуйте свой ответ.

    __name__ - это специальная переменная, которая содержит имя текущего модуля. Выполнение программы начинается с главного или кода с нулевым отступом. Таким образом, в данном случае __name__ имеет значение __main__. Если файл импортируется из другого модуля, __name__ содержит имя этого модуля.

  15. Объясните функции join() и split() в Python.

    Функция join() может быть использована для объединения списка строк на основе разделителя в одну строку. Функция split() может быть использована для разделения строки на список строк на основе разделителя.
