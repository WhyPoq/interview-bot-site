  1. Что такое Type hinting в PHP?

    Type hinting - это функция в PHP, которая позволяет разработчикам указывать ожидаемый тип данных переменной или параметра функции. Это помогает гарантировать, что код будет работать как ожидается, когда данные непредвиденного типа передаются.

  2. Как установить бесконечное время выполнения для PHP скрипта?

    Мы добавляем set_time_limit(0) в начало скрипта, чтобы сделать время выполнения бесконечным. Это помогает предотвратить ошибку PHP 'превышено максимальное время выполнения'. Мы также можем указать это в файле php.ini.

  3. Как экспортировать данные в файл Excel?

    Обычно данные получают в формате, поддерживаемом Excel. Например, мы можем записать файл .csv, выбрать разделитель между полями (например, запятую), а затем открыть файл в Excel.

  4. Как PHP может взаимодействовать с HTML?

    PHP предназначен для взаимодействия с HTML. Возможно встраивание PHP скриптов на HTML страницу без проблем и передача информации из HTML в PHP.

  5. Как PHP обрабатывает многобайтовые символы?

    PHP имеет встроенные функции и библиотеки для работы с многобайтовыми кодировками, такими как UTF-8. Функции, такие как mb_strlen(), mb_substr() и mb_convert_encoding(), обычно используются для манипуляций с многобайтовыми строками. Кроме того, расширение mbstring обеспечивает полную поддержку операций со строками многобайтовых символов.

  6. Что такое NULL?

    NULL - это особый тип значения в PHP. Он представляет собой переменную без значения и обычно используется для обозначения отсутствия сделки.

  7. Какие два наиболее распространенных способа начать и закончить блок кода PHP?

    При написании блока кода PHP два наиболее распространенных способа начать и закончить его - использовать открывающий тег.

  8. Назовите некоторые функции в PHP.

    Некоторые из наиболее часто используемых функций в PHP включают strlen(), str_replace(), urlencode() и md5(). Эти функции могут выполнять различные задачи, такие как подсчет количества символов в строке, замена содержимого строки, кодирование URL-строки и генерация хеш-значения.

  9. Каково использование обратного вызова в PHP?

    Обратный вызов - это PHP функция, используемая для выполнения других частей кода, и часто используется для добавления пользовательской функциональности или для предоставления альтернативного способа выполнения кода.

  10. В чем разница между исключением времени выполнения и исключением времени компиляции?

    Исключения времени выполнения возникают во время выполнения кода и вызваны неожиданными событиями, такими как попытка доступа к индексу массива, которого не существует. Исключения времени компиляции, с другой стороны, возникают до выполнения кода и обычно вызваны кодом, который не компилируется правильно, таким как синтаксические ошибки или попытка использовать необъявленную переменную.

    Для достижения этого результата нам нужно использовать print_r().

  12. Что мы подразумеваем под стандартом PSR? Какова его важность?

    Поскольку существуют различные стандарты кодирования между разработчиками и компаниями, можно столкнуться с проблемами при попытке исправить код другого разработчика, который может иметь другую структуру. Вот где на помощь приходит стандарт PSR. Он стандартизирует внешний вид кода, что помогает устранить проблемы и даже синтаксические ошибки в некоторых случаях.

  13. Как передать данные с одной PHP-страницы на другую?

    Данные могут быть переданы с одной PHP-страницы на другую с помощью методов, таких как параметры URL, отправка формы, сессии, куки или путем хранения данных в базе данных и извлечения их на последующей странице.

  14. Объясните концепцию сессий в PHP. В чем отличие от cookies?

    Сессии в PHP - это способ сохранения данных между несколькими запросами для одного пользователя. В отличие от cookies, которые хранят данные на стороне клиента, сессии хранят данные на стороне сервера. Сессии обычно более безопасны, так как данные не доступны клиенту.

  15. Что нужно для использования функции изображения?

    Для выполнения функций изображения необходима библиотека GD, которая встроена в PHP.
